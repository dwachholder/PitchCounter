PitchCounter
============

A simple Pitch Counter that will give you and your group (e.g., students) support in doing presentations in the style of an elevator pitch.



Keyboard Commands
======================
[BLANK] = Start/Stop
[ESC] = Reset

Touch Commands
======================
[SINGLE-TAP] = Start/Stop
[DOUBLE-TAP] = RESET
[SWIPE] = RESET
